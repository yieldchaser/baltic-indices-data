## Main
in Weekly Vessel Valuations Report 23/07/2025
Image reference: 2025-07-23_weekly-vessel-valuations-report-july-22-2025_img1.png
Bulkers: Bulker values have remained mostly stable this week, with an increase in smaller tonnagedue to sales such as Sea Dolphin C.
Capesize BC Pacific North (180,300 DWT, Mar 2011, Dalian Shipbuilding Industry Corp) sold SS/DD due to unknown Greek buyers for USD 25.3 mil, VV Value USD 26.62 mil
Panamax BC White Whale (76,000 DWT, Jul 2012, Hudong Zhonghua) sold DD due to unknown Greek buyers for USD 12.75 mil, VV Value USD 14.03. mil
Handy BC Sea Dolphin C (33,800 DWT, Dec 2011, 21st Century) sold to unknown Vietnamese buyers for USD 11.7 mil, VV Value USD 11.31 mil
Tankers: Tanker values remaining mainly stable last week as vintage VLCCs see an uptick based on recently confirmed deals.
VLCC Atlantic Loyalty (307,300 DWT, Apr 2007, Dalian Shipbuilding Industry Corp) sold DD Passed to undisclosed buyers for USD 44 mil, VV Value USD 44 mil
VLCC City of Tokyo (304,000 DWT, Mar 2004, Universal) sold to undisclosed buyers for USD 41.5 mil, VV Value USD 36.22 mil
MR2 (Chemical/Product) Oriental Diamond (50,800 DWT, Apr 2008, SPP) sold to undisclosed buyers for USD 16 mil, VV Value USD 16.23 mil
Containers: Values are up for the smaller segments following recent sales.
Panamax Spring C (3,450 TEU, Jan 2007, Hyundai Mipo) sold to CMA CGM for USD 28 mil, VV Value USD 27.7 mil
Feedermax Vega Alpha (917 TEU, Nov 2005, Volharding) sold to buyers from the UAE for USD 8 mil, VV Value USD 8.3 mil.
Feedermax Shecan (908 TEU, Oct 2008, Nanjing Wujiazui) sold to Middle Eastern buyers for USD 9.5 mil, VV Value USD 9.1 mil.
