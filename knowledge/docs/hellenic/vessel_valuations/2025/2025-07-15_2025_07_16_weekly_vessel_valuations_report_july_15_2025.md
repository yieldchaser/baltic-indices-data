## Main
in Weekly Vessel Valuations Report 16/07/2025
Image reference: 2025-07-16_weekly-vessel-valuations-report-july-15-2025_img1.png
Bulkers: Bulker values have remained mostly stable this week, with an increase in older smallertonnage due to sales such as Rijn Confidence and Jin Gang
Capesize BC Pacific South (176,000 DWT, Jun 2012, Jiangsu Rongsheng) sold DD due to undisclosed buyers for USD 27.1 mil, VV Value USD 27.57 mil
Kamsarmax BC Ultra Puma (81,900 DWT, Aug 2016, Tsuneishi Zosen) sold to undisclosed buyers for USD 25 mil, VV Value USD 25.45 mil
Supramax BC Pan Rapido (56,900 DWT, Jul 2011, COSCO Zhoushan) sold to unknown Chinese buyers for USD 11.4 mil, VV Value USD 12.1 mil
Supramax BC Jin Gang (55,900 DWT, Jun 2009, Shanghai Shipyard) Sold TC Back to Huwell Shipping for USD 10.8 mil, VV Value USD 9.57 mil
Handy BC Rijn Confidence (33,300 DWT, Jun 2013, Shin Kurushima Onishi) sold to undisclosed buyers for USD 14.3 mil, VV Value USD 13.7 mil
Tankers: Tanker values remained stable for the larger tonnage with some downward movements for Suezmax and LR1's coinciding with a drop for Suezmax rates
LR2 Hesperia Tide (115k DWT, Jul 2025, Zhoushan Changhong) sold for USD 70 mil to New Shipping Ltd, VV value USD 68.56 mil
Aframax Ise Princess (105,400 DWT, Jul 2009, Sumitomo) sold for USD 32.50 mil, VV value USD 32.34 mil
MR2 PS Vancouver (51k DWT, Jan 2008, STX Offshore) sold for USD 13 mil with forward delivery, VV value USD 15.87 mil
Containers: Post panamax midage values down following a recent sale, and handy container values up in older vessels. Other sectors remain stable with limited sales candidates this week
Post Panamax Hakata Seoul (8,540 TEU, Aug 2010, Mitsubishi HI) sold to MSC for USD 75 mil, VV Value USD 77.6 mil
Handy Container A Obelix (1,702 TEU, Sep 2008, Wadan Yards MTW) sold to Heidmar for USD 25.25 mil inc TC, VV Value USD 16 mil.
