Handy BC Kalixenos (33,700 DWT, Apr 2010, Samjin Shipbuilding Industries) sold to undisclosed buyers for USD 10.90 mil, VV Value USD 11.70 mil - DD Due.
Tankers : Mid age Suezmax tankers values have softened, whilst older MR tankers have firmed
Suezmax Ridgebury Elizabeth B and Ridgebury Lessley B (158,600 DWT, Jun 2011-Sep 2013, Hyundai Heavy Ind, Samsung) sold to Navig8 for USD 101.50 mil, VV Value USD 108.36 mil - On Subs, SS/DD Due
Aframax Nectar Sea (105,400 DWT, Apr 2008, Sumitomo) sold to undisclosed buyers for USD 37.00 mil, VV Value USD 38.06 mil - SS/DD Due.
MR2 Ridgebury John B (46,000 DWT, Jan 2007, Shin Kurushima Onishi) sold to undisclosed buyers for USD 23.50 mil (On Subs), VV Value USD 18.25 mil - On Subs.
MR1 (Chem / Product) Norient Saturn (40,400 DWT, Dec 2007, Santierul Naval Constantza) sold to Sea Transport Group for USD 19.65 mil, VV Value USD 17.51 mil - SS/DD Passed.
Containers : Container values have remained stable
