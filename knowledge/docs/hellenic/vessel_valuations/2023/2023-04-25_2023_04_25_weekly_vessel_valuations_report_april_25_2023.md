## Main
in Weekly Vessel Valuations Report 25/04/2023

Bulkers : Bulker values have remained stable
Panamax Valiant Spring (81,900 DWT, Dec 2015, Tsuneishi Zhoushan) sold to undisclosed buyers for USD 28.30 mil, VV Value USD 26.94 mil - BWTS.
Ultramax CL Ebisu (61,300 DWT, Nov 2014, Iwagi Zosen) sold to Phoenix Bulk Carriers for USD 26.70 mil, VV Value USD 24.96 mil - BWTS.
Supramax Seacon Qingdao (56,500 DWT, Mar 2013, Taizhou Sanfu Shipbuilding) sold to unknown Chinese buyers for USD 15.67 mil, VV Value USD 15.11 mil - SS/DD Passed.
Supramax Worldera 3 (50,300 DWT, Jul 2000, Mitsui Tamano) sold to undisclosed buyers for USD 8.00 mil, VV Value USD 8.87 mil - DD Passed.
Tankers : LR values have firmed, whilst VLCC values have softened in the past week
VLCC Princess Mary (306,200 DWT, Jun 2004, Mitsubishi HI) sold to unknown Chinese buyers for USD 51.00 mil, VV Value USD 51.00 mil - BWTS.
LR2 Pro Alliance (105,300 DWT, Sep 2008, Hyundai Heavy Ind) sold to undisclosed buyers for USD 37.00 mil, VV Value USD 40.82 mil - SS/DD Due.
LR1 Hafnia Rhine (76,600 DWT, Mar 2008, Dalian Shipbuilding Industry Corp) sold to Quantum Global for USD 26.50 mil, VV Value USD 23.90 mil - SS/DD Passed.
LR1 Pike (73,700 DWT, Oct 2008, New Times Shipbuilding) sold to undisclosed buyers for USD 22.50 mil, VV Value USD 23.57 mil - SS/DD Due.
MR2 Kiribora (50,000 DWT, Jan 2013, SPP) sold to Union Maritime for USD 33.50 mil, VV Value USD 34.10 mil - SS/DD Passed.
Containers : Older tonnage Post Panamax values have softened after the sale of Lisbon, all other values have remained stable
No Reported sales this week.
