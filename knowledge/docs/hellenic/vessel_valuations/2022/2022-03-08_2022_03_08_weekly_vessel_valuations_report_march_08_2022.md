## Main
in Weekly Vessel Valuations Report 08/03/2022
Image reference: 2022-03-08_weekly-vessel-valuations-report-march-08-2022_img1.png
Tanker: Modern Tanker values have firmed.
VLCCs Nautilus and Navarin (307,300 DWT, Dec 2006 & Apr 2007, Dalian Shipbuilding Industry Corp) sold to Sinokor in an en bloc deal for USD 63.50 mil, VV en bloc value USD 61.90 mil.
Aframax Axel Spirit (115,400 DWT, Apr 2004, Samsung) sold to Chinese buyers for USD 13.20 mil, VV value USD 13.40 mil - DD Due.
LR2 Aquanus (109,700 DWT, Oct 2006, Dalian Shipbuilding Industry Corp) sold for USD 12.50 mil, VV value USD 13.94 mil - SS/DD Due.
MR1 Duke I (40,100 DWT, Feb 2002, Hyundai Mipo) sold for USD 7.10 mil, VV value USD 6.13 mil - SS/DD Passed.
MR1 Torm Tevere (37,400 DWT, Jul 2005, Hyundai Mipo) sold to Nigerian buyers for USD 8.30 mil, VV value USD 8.65 mil.
Bulker: Bulker values have remained stable.
Panamax Darya Kirthi (80,500 DWT, Feb 2012, STX Offshore) sold to Greek buyers for USD 25.00 mil, VV value USD 23.14 mil - SS/DD Due.
Panamax Coral Ruby (76,600 DWT, Jun 2008, Imabari) sold for USD 18.00 mil, VV value USD 18.16 mil.
Supramax Orchid (55,600 DWT, Feb 2012, Mitsui Tamano) sold to Greek buyer for USD 22.60 mil, VV value USD 21.22 mil.
Handy Bulker ES Mercury (32,200 DWT, Aug 2008, Kanda) sold for USD 15.50 mil, VV value USD 15.25 mil.
Container: Container values have firmed.
No sales to report.
