Bulkers: Panamax and Handy values have softened

Supramax Bonita (58,100 DWT, Sep 2010, Tsuneishi Zhoushan) sold to unknown Greek buyers for USD 15.80 mil, VV Value USD 15.52 mil – BWTS.

Handy BC African Venture (34,700 DWT, Jun 2012, Chengxi Shipyard) sold to undisclosed buyers for USD 12.90 mil, VV Value USD 13.41 mil – SS/DD Passed.

Handy BC AS Elenia (34,400 DWT, Sep 2011, SPP) sold to undisclosed buyers for USD 13.30 mil, VV Value USD 13.63 mil – BWTS.

Tankers: Tanker values have remained stable

Aframax Atlantis (105,400 DWT, Sep 2008, Sumitomo) sold to unknown Greek buyers for USD 38 mil, VV Value USD 38.56 mil – BWTS.

MR2 Markos I (46,000 DWT, Sep 2005, Uljanik Brodogradiliste) sold to unknown Chinese buyers for USD 17.00 mil, VV Value USD 19.66 mil – BWTS.

Containers: Container values have softened

No reported sales this week.
